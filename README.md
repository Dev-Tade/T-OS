# T-OS

**An Open Source Operating System:** <br />

### PREREQUISITES

0. Linux | WSL
1. MAKE
2. NASM
3. GCC
4. LD
5. MAKE

### BUILDING

```console
$ make run
```

### MOTIVATION

**T-OS was a side project with these objectives:**

1. Bootloader (Assembly)
2. Kernel (C)

**T-OS originally was called c-OS (Custom - OS)**<br/>

**T-OS is heavily inspired by Linux, BSD, UNIX**<br/>
